# Venezuela Earthquake Outreach Dataset

Public static download page for outreach datasets related to the Venezuela
earthquake aid app.

## Files

- `data/venezuela-earthquake-contact-dataset.csv`: prioritized public contact
  dataset with source cross-checks, public contact surfaces, confidence scores,
  risk notes, and next steps.
- `data/venezuela-earthquake-influencer-dataset.csv`: broader diffusion list
  with media, creators, diaspora hubs, humanitarian organizations, and community
  businesses.

## Privacy note

This repository must not be used as a people-finder. The datasets intentionally
avoid cedula numbers, personal RIFs, private addresses, voter-center data, health
data, private phone numbers, and other sensitive personal data. Cedula.com.ve is
listed only as a possible technical/API partnership contact, not as a source for
identifying or contacting citizens.
